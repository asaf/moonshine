Moonshine

About

Moonshine is an application framework using Google Guice's dependency injection as its foundation. It is based on the notion of Service. Services implement some functionality, they can depend on each other and they can register bindings in Guice

Moonshine provides mechanisms for configuring, executing and testing the services. It also comes out of the box with ready to use services for database access (H2, HSQLDB), servlets and web sockets (Jetty, Tomcat), JTA (Atomikos), JPA (Hibernate), Spring Data, JMX, JAX-RS (Jersey), and many more.
