Introduction to Moonshine

Overview

Moonshine is a Java application framework. It is lightweight, but at the same it integrates technologies and libraries needed for modern application development. Whether you are developing console program, web application or enterprise-ready backend service, modular architecture of Moonshine allows you to built solution that matches complexity of your problem.

Service

Moonshine is based around the notion of a Service - a distinct component of functionality - and forces you to also modularize your application in the same way. Services can then be created and configured at runtime using flexible configuration file.

Moonshine out of the box provides services to start and configure transactions, database access with migrations, persistence using Hibernate and Spring Data, two HTTP engines Jetty and Tomcat, servlets, websockets and REST services. Additionally Moonshine contains services which allows to profile your application and retrieve metrics from it using JMX directly or through the provided REST wrapper or web console.

Testing

Moonshine is built from the ground up with the high focus on testability and test-driven development. Special support for JUnit allows your tests to be executed inside Moonshine container. Network services were carefully written to always start on random port, so you can safely execute your tests in parallel. There is also support for mocking and testing web applications.
